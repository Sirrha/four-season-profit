Firebase-oppsett – gjøres én gang
1
Gå til console.firebase.google.com og logg inn med en Google-konto.
2
Klikk «Add project», gi det et navn (f.eks. four-season-as), og opprett prosjektet.
3
I prosjektet: klikk «Build» → «Firestore Database» → «Create database» → velg «Start in test mode» → velg en region nær Norge (f.eks. europe-west3).
4
Gå til Project Settings ⚙️ (tannhjulet øverst til venstre) → «Your apps» → klikk </> (Web app) → registrer appen → kopier firebaseConfig-objektet som vises.
5
Lim inn hele firebaseConfig-objektet nedenfor (inkludert krøllparentesene { }):

⚠️ Viktig – Firestore-regler: I Firestore Console → Rules, bytt ut innholdet med disse reglene og publiser:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }

Dette er OK for internt bedriftsbruk. Du kan legge til innlogging senere.
